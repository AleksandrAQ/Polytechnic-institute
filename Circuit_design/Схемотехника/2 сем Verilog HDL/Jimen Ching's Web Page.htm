Last Updated 2000 April 15

Welcome to the VBS project homepage. VBS stands for Verilog Behavioral Simulator. This was a software I wrote for my senior design project at the University of Hawaii. Since then, I have extensively added more features. This homepage is a depository of new ideas and updates of changes to VBS.

Version 1.3.7 of VBS is now released. See the README file below for details.

Below are more links about VBS.

  • VBS is now hosted by SEUL and is part of the gEDA project. The gEDA-dev mailing list will be used for discussions of VBS. It is shared by other applications that are part of the gEDA project. To join the mailing list, send an email to majordomo@geda.seul.org, with 'subscribe geda-dev' in the body.
  • VBS can be downloaded from CVS. Use the following commands: 
    cvs -d :pserver:anoncvs@cvs.seul.org:/home/cvspsrv/cvsroot login
Enter 'guest' as the password.
cvs -d :pserver:anoncvs@cvs.seul.org:/home/cvspsrv/cvsroot co vbs
    If you need more information, visit the CVS webpage.

Misc:

  • I am the new maintainer of a cross-platform Win32 API library. The library is called TWIN and you can visit Willows , the company that owns this library. I plan to add multimedia support to the list of supported APIs. I should also mention that this library comes with a Windows emulator. It can run on any platform that TWIN supports. For platforms that does not support the binary emulator, here are some sample sources to link with the library. The current CVS tree supports building the MFC library.
  • I also maintain the re-release of the VPP software package by Hemi Thaker. The new re-release is called VBPP version 1.0.3. You can take a look at the README file for details.
